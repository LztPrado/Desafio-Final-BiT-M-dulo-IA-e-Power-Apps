# Desafio-Final-BiT-M-dulo-IA-e-Power-Apps
Desafio Final do módulo de IA e Power Apps. Este projeto foi desenvolvido para simular um cenário real, exigindo análise crítica e definição de problema. A proposta é criar uma aplicação em Power Apps para estruturar e analisar o processo de atendimento de um comércio local, transformando interações em dados estratégicos.
